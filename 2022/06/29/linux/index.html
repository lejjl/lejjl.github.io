linux学习

2022-06-29

命令

  • pwd 查看当前所在目录

  • ls 查看目录下文件

  • ls -l 查看目录下文件的信息（列表形式）

  • ls -l -h 按照kb显示文件大小（不加-h默认为字节显示）

  • ls -a 查看所有文件夹（包括隐藏文件夹）

  • cd 打开目录，进入文件夹

  • cd .. 返回上一级目录

  • cd ~ 回到根目录

  • mkdir 创建文件目录

  • rm -rf 直接删除文件目录(如执行rm -rf test2回车就删除test2了）

  • touch 创建文件命令(如执行touch test.txt 创建了一个名为test文件）

  • mv 移动文件 (如执行mv test.txt aa表示移动test.txt文件去aa文件夹) (剪切)

  • cp 复制文件 (如执行cp test.txt aa表示复制test.txt文件去aa文件夹) (复制)

  • cp -r复制文件夹 (如执行cp -r bb aa表示复制bb文件夹去aa文件夹) (复制)

  • which 查看命令所在位置（如执行which ls表示查看ls 哪个文件夹）

  • clear 清屏

  • date时间

  • rpm -ql nginx 查询nginx 相关全部文件

ls通配符

  1. ls *

  2. ls ?

  3. ls []

cd用法

  1. cd . 当前目录
  2. cd .. 返回上一级
  3. cd - 返回上一级
  4. cd ../.. 返回上两级 以此类推
  5. cd 文件名 进入输入的类名
  6. cd ~ 返回根目录

rm用法

  1. rm 文件名 删除某个文件
  2. rm -d 文件夹名称 删除某个文件夹
  3. rm -r 文件夹名称 删除某个文件夹 与rm -d 一样
  4. rm -rf 删除当前目录下的所有文件及目录，并且是直接删除，无需逐一确认

cat/more用法

  1. cat 文件 进入文档
  2. cat -b 文件 进入文档并且前面会加标号（空格除外）
  3. cat -n 文件进入文档并且前面会加标号（包括空格）
  4. cat etc/group 进入用户组
  5. more 文件 进入文档 分屏显示

grep用法

  1. grep 需要搜索内容 搜索的文件 文件内容的搜索
  2. grep -n 需要搜索内容 搜索的文件 文件内容的搜索 并展示在多少行
  3. grep -v 需要搜索内容 搜索的文件 文件内容的搜索展示不包含搜索内容的全部内容
  4. grep '#' 表示搜索内容中包含#的内容
  5. grep ^ '#'表示以#开头的内容 (^表示以什么开头）

vi/vim用法

  1. 输入vi/vim 进入命令模式

mv/cp 用法

  1. cp 复制文件 (如执行cp test.txt ../aa表示复制test.txt文件去上一级aa文件夹) (复制)

  2. cp -r复制文件夹 (如执行cp -r bb ../aa表示复制bb文件夹去上一级aa文件夹) (复制)

  3. mv 移动文件 (如执行mv test.txt ../aa表示移动test.txt文件去上一级aa文件夹) (剪切)

  4. 特殊情况

find 精确查找

  1. find -name test 全局查找带有test字母的文件
  2. find -iname 'test' 模糊搜索 文件中含有test字母的文件 不区分大小写（-i 表示不区分大小写）
  3. find -size +1 为1kuai=512byte +1为大于512k的文件 -1为小于512k的文件 =1的时候为等于512k的文件 =号可以省略
  4. find -user root 表示查找用户名为root的人创建的文件
  5. find -mmin -5表示5分钟内哪些文件被修改了 -5表示五分钟以内 +5表示5分钟以外 5表示5分钟

echo回显

  1. echo xxxxx 会在控制台回显 输入啥显示啥
  2. echo hello >text.txt hello表示输入内容， >一个右尖括号表示覆盖， text.txt表示需要修改的文件
  3. echo hello >>text.txt hello表示输入内容， >一个右尖括号表示添加， text.txt表示需要修改的文件
